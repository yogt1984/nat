# Spannung Experiment — Final Results

**Date**: 2026-05-14
**Duration**: Phases A–C, single session
**Status**: Concluded — signal validated, not tradeable as standalone

## Discovery

L1 book imbalance (`imbalance_qty_l1`) predicts 1–5s forward returns with
IC=0.44–0.48 across BTC, ETH, SOL. This is ~10x typical quant feature ICs.

## Key Numbers

| Metric | Value |
|--------|-------|
| Best IC (tick-level) | 0.479 (BTC), 0.484 (ETH), 0.441 (SOL) |
| IC stability | 100% positive windows over 10h (94 non-overlapping 5-min windows) |
| OOS replication | IC 0.42–0.47 on 2026-05-11 (24h), 0.41–0.45 on 2026-05-10 |
| Lag decay | 0.48 → 0.39 (0.1s) → 0.33 (1s) → 0.18 (5s) → 0.08 (10s) |
| Per-trade edge | 0.17–0.37 bps |
| Taker round-trip cost | 7.0 bps |
| Cost-to-edge ratio | ~20x |
| Best regime-gated IC | 0.51 ("low entropy + high illiquidity") |
| Bar-level IC (aggregated) | 0.05–0.20 (2–3x degradation) |

## Phases

### Phase A: Grid Search
- 1,350 combos per symbol (3 flow × 3 illiq × 6 alpha × 5 beta × 5 horizons)
- Raw imbalance_l1 dominates — Spannung EWM formulation hurts by 8–10 IC points
- Best config: alpha=0.5s, beta=60s, imbalance_qty_l1 / illiq_composite
- Go/no-go gate (IC > 0.05): **PASSED** at 6–8x threshold

### Phase A.2: Bias Check
- Smooth lag decay confirms causal signal, no look-ahead bias
- EWM and illiquidity denominator both reduce IC — raw signal is best

### Phase A.3: Out-of-Sample
- 4 dates tested (2026-05-12 IS, 2026-05-11/10/04-25 OOS)
- IC stable at 0.42–0.48 on recent data, 0.27–0.36 on older data
- 2026-05-07/08 data frozen (old ingestor bug) — excluded

### Phase B: Cost-Aware Backtest
- Every threshold loses money after taker and maker fees
- Regime gating: directionally correct (+6% IC) but insufficient (20x gap)

### Phase C: Longer-Horizon Sweep
- 6 signal variants × 6 timeframes (30s–15min) × 4 forward horizons
- IC degrades catastrophically with aggregation
- Zero overlap between IS and OOS profitable combos — noise on thin data

## Conclusions

1. **L1 book imbalance is a genuine, persistent microstructure signal** — strongest
   single feature in the 191-feature vector
2. **Spannung formulation adds no value over raw imbalance** at any timescale tested
3. **Not tradeable as standalone** at retail fees — 20x cost-to-edge gap at ticks,
   IC collapses at bars
4. **Primary value**: top input feature for multi-signal models at lower frequency

## Tools Built

- `nat spannung` — grid search across parameter space
- `nat spannung backtest` — cost-aware backtest + regime gating
- `nat spannung horizon` — longer-horizon bar sweep (30s–15min)
