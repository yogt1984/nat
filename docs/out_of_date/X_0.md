## Decision Gates

### After 2 Weeks Data
Run `make test_hypotheses`:
- **GO (4-5 pass):** Proceed to paper trading
- **PIVOT (2-3 pass):** Focus on validated signals only
- **NO-GO (0-1 pass):** Continue data collection, refine hypotheses

### After Regime Features
Run cluster analysis:
- **4-6 distinct clusters:** Proceed with classifier
- **< 4 clusters:** Revisit feature definitions

### After Paper Trading
2 weeks forward test:
- **Sharpe > 0.5, OOS/IS > 0.7:** Consider live deployment
- **Otherwise:** Iterate on model/features
