## Setup: Isolate Healthy Data

Copy only the healthy post-restart files into a clean directory so all scripts
operate on valid data only:

```bash
mkdir -p data/features/2026-05-12-clean
cp data/features/2026-05-12/20260512_06441*.parquet \
   data/features/2026-05-12/20260512_0[7-9]*.parquet \
   data/features/2026-05-12-clean/
```

Use `DATA=data/features/2026-05-12-clean` as the data directory for all commands below.
